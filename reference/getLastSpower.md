# Get previously evaluated Spower execution

If the result of
[`Spower`](https://philchalmers.github.io/Spower/reference/Spower.md) or
[`SpowerBatch`](https://philchalmers.github.io/Spower/reference/Spower.md)
was not stored into an object this function will retrieve the last
evaluation.

## Usage

``` r
getLastSpower()
```

## Value

the last object returned from
[`Spower`](https://philchalmers.github.io/Spower/reference/Spower.md) or
[`SpowerBatch`](https://philchalmers.github.io/Spower/reference/Spower.md)
